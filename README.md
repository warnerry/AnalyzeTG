# Telegram Content Analyzer 

Анализатор контента Telegram-канала профбюро для улучшения вовлечённости и качества публикаций.

## Возможности
- Сбор постов канала
- Подсчёт вовлечённости (engagement)
- Подготовка данных для ML-анализа (темы, форматы, время публикаций)

## Установка
```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt 
```
